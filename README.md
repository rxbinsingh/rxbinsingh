## Projects

**VIPER** — Deepfake detection via identity displacement reaction (AB + C → AC + B)  
CLIP ViT-L/14 frozen backbone + identity-anchored face preprocessing  
0.991 AUC · 95.2% accuracy · 4s per video · trained on 530 videos in 25 min  
[GitHub](https://github.com/rxbinsingh/VIPER) · [HuggingFace](https://huggingface.co/rxbinsingh/VIPER) · [Live Demo](https://huggingface.co/spaces/rxbinsingh/VIPER)

**SynID** — Zero-shot identity-consistent image generation from text alone  
No real reference photos. No dataset. No pretraining. ~5 min on T4 GPU.  
0.954 CLIP identity · 0.791 ArcFace · outperforms IP-Adapter FaceID  
[GitHub](https://github.com/rxbinsingh/SynID) · [HuggingFace](https://huggingface.co/rxbinsingh/SynID) · [Paper](https://doi.org/10.13140/RG.2.2.30671.85925)

**GHOST** — Zero-shot supply chain disruption forecasting without labeled data  
Closed-loop self-distillation: BiLSTM + dual-level GNN + probe-based drift correction  
MSE 0.0081 · 30k+ high-risk cases detected · 180k+ real orders · 15 min on T4 GPU  
[GitHub](https://github.com/rxbinsingh/GHOST) · [HuggingFace](https://huggingface.co/rxbinsingh/GHOST) · [Paper](https://doi.org/10.13140/RG.2.2.27961.94567)
